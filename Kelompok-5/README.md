# 🗺️ Map Banda Aceh - Marker Lokasi

## 📌 Deskripsi

Project ini merupakan aplikasi peta interaktif berbasis **Leaflet.js** yang menampilkan berbagai lokasi di Banda Aceh seperti:
* 🏨 Hotel / 🏡 Homestay
* 🏫 Sekolah
* 🏢 Dinas

Setiap lokasi ditampilkan dalam bentuk marker dengan ikon yang berbeda sesuai kategori.

## 🎯 Fitur

* Menampilkan peta Banda Aceh
* Menampilkan marker dari file JSON
* Popup informasi (nama lokasi + link Google Maps)
* Ikon berbeda berdasarkan jenis lokasi:
  * 🏨 Hotel / 🏡 Homestay
  * 🏫 Sekolah
  * 🏢 Dinas
* Data diambil secara dinamis menggunakan JavaScript `fetch()`

## ⚙️ Cara Menjalankan

1. Download / clone repository
2. Buka folder project di VS Code
3. Jalankan dengan Live Server
   (klik kanan `index.html` → **Open with Live Server**)

## 🗺️ Teknologi yang Digunakan

* HTML
* CSS
* JavaScript
* Leaflet.js
* OpenStreetMap
